# ASCII Generator

## About 
Python program to convert images into ASCII art.

## Getting Started

### Prerequisites

```
numpy==1.16.2
Pillow==7.1.2
```

### Installing

Install numpy
```
pip install numpy
```
Install Pillow
```
pip install Pillow
```

## Usage

1. Assign path of image in `FILE` variable in `source.py`
2. Run: `python source.py`
